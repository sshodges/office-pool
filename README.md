# office-pool

## Setup

```shell
$ git clone https://github.com/sshodges/office-pool.git
$ npm install
$ cd client
$ npm install
```

## Start Application

Run Backend & Front-end

```shell
$ npm run dev
```

Run Backend Only

```shell
$ nodemon server.js
```

Run Front-end Only

```shell
$ npm run client
```
